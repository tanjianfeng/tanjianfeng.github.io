# Tanjianfeng's Professional Portfolio

This is my personal website hosted on GitHub Pages, showcasing my professional experience, skills, and projects in a CV/resume format.

## About

This website serves as my online CV and professional portfolio, providing information about:
- Professional Summary
- Technical Skills
- Work Experience
- Education
- Notable Projects
- Contact Information

## Structure

- `index.html` - Main CV page with professional information
- `stylesheets/` - CSS styling including CV-specific components
- `images/` - Profile photo and other images
- `javascripts/` - JavaScript files

## Design

The site uses a clean, professional CV layout with:
- Responsive design for mobile and desktop
- Organized sections for easy navigation
- Professional color scheme
- Modern typography and spacing

## Hosting

This site is hosted on GitHub Pages at: https://tanjianfeng.github.io

## Updates

To update content, edit `index.html` and commit changes. GitHub Pages will automatically rebuild and deploy the site.
